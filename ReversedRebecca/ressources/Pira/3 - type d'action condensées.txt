////// NAME //////



Le Type d'Action NAME permet de renommer un objet.

exemple :
TIME -> 0 = NAME -> BOB
exemple d'écriture condensée :
TIME -> 0 = GOTO -> NAME:BOB

Lorsque l'état sera vérifié, l'objet sera renomé Bob.

Cette fonction est très utile juste après la génération d'une map afin d'établir le nom d'un personnage (et afficher les bonnes textures).


////// FACING //////


Le Type d'Action FACING permet de changer l'orientation d'un objet.

exemple :
TIME -> 0 = FACING -> RIGHT
exemple d'écriture condensée :
TIME -> 0 = GOTO -> FACING:RIGHT

Lorsque l'état sera vérifié, l'objet regardera à droite.

Il est tout à fait possible, au lieu d'écrire une direction, d'écrire le nom d'un autre objet pour que l'objet se tourne vers direction.
ex : 
TALK -> COMPUTER = FACING -> BOB
À la fin du dialogue "COMPUTER", l'objet se tournera vers BOB.

Attention ! Il est facile de se tromper : c'est seulement l'orientation de l'objet dans lequel est écrite cette Inscription qui change ! Si on veut changer le facing du player ou d'un npc, il faut écrire cette Instruction dans LEURS fichiers.


////// MOVE //////


Le Type d'Action MOVE permet définir un GOAL à un objet. S'il peut se déplacer, il se déplacera dans sa direction.

exemple :
TIME -> 0 = MOVE -> 33-35
exemple d'écriture condensée :
TIME -> 0 = GOTO -> MOVE:33-35

Lorsque l'état sera vérifié, l'objet cherchera à sen rendre en 33 35

Attention ! Les coordonnées négatives sont impossibles en Pira. Veillez à bien choisir l'origine (en haut à gauche par exemple)


////// SLOW //////


Le Type d'Action SLOW fait exactement la même chose que MOVE, mais si l'objet est un NPC, il se déplacera plus lentement.

exemple :
TIME -> 0 = SLOW -> 33-35
exemple d'écriture condensée :
TIME -> 0 = GOTO -> SLOW:33-35


////// TP //////


Le Type d'Action TP téléporte un objets aux coordonées indiquées.

exemple :
TIME -> 0 = TP -> 33-35
exemple d'écriture condensée :
TIME -> 0 = GOTO -> TP:33-35

Lorsque l'état sera vérifié, l'objet sera téléporté en 33 35



Écriture condensée : oui.
